# stdlib.h

## 简介：

    一些常用函数
## 函数列表：

    ``void *malloc(ssize_t size)`` ： 普通的 ``malloc``

    ``void free(void *ptr)`` : 释放内存

    ``int abs(int x)`` : x 的绝对值

    ``long labs(long x)`` : x 的绝对值

    ``long long llabs(long long x)`` : x 的绝对值

    ``int atoi(const char *str)`` 字符串转数字

    ``void exit(int status)`` : 普通的 ``exit``
